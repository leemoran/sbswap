slidebars swap

Extends Adam Smith's Slidebars to display multiple content panels.

1
2
3
Download lee.moran.graphics GitHub

1

Install Slidebars as usual. Include slidebars.swap.js after Slidebars.

1 2 3

2

In your sb-slidebar div, place each panel's content in a div with the class sb-swap-panel and an ID of whatever you want.

Example:

            
<div class="sb-slidebar sb-right sb-style-overlay">
  <div class="sb-swap-panel" id="one">Content for panel one</div>
  <div class="sb-swap-panel" id="two">Content for panel two</div>
</div>
1 2 3

3

Add data-sbswap="theID" to any element that you want to toggle a Slidebars panel, where theID is the ID of the panel you want it to toggle.

For styling, Slidebars Swap will add the class sb-active-control to any elements that control the currently open panel.

  • lee.moran.graphics
  • Slidebars Swap is compatible with all of Slidebars' options and animation styles.
  • Any type of content that works with a base Slidebars install can be added to a Slidebars Swap panel.
  • No modifications to Slidebars' core Javascript or CSS are necessary for Slidebars Swap to work.
